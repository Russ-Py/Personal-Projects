# Quantum Computing Projects

## Analysis of Grover's Algorithm

For this project, I took my independent simulation of Grover's Algorithm and applied it to my mathematics capstone. The goal is to expand on my theoretical understanding of the algorithm and simulate it with four error models. I simulated the algorithm under thermal relaxation error, readout error, and depolarization error. Of these, the depolarization error proved to affect search accuracy the most, as Grover's relies on information stored in the probability amplitudes of the system's state vector.

Included in the folder for this project is the Jupyter notebook for the simulation and the paper I wrote to top my mathematics undergraduate experience.

## Encrypted State Cloning

It has been shown that one can entangle a qubit, A, with a set of qubits, each comprising one half of a Bell pair. This is referred to as "encrypted cloning" and does not violate the no-cloning theorem, as the information in qubit A's wavefunction is only retrievable from one qubit. I simulated this process and wrote up a paper for my physics capstone. 

In the folder for this project, you will find the Jupyter notebook for this algorithm, the paper I wrote, and all diagrams I created to illustrate this phenomenon.
